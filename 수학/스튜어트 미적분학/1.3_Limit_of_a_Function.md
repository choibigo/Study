# 직관적인 극한
![](https://miro.medium.com/max/1400/1*sE3VHxC5n7iyaTfRFZm61w.png)
- f(x)가 있을떄 x가 a로 갈떄 f(x)는 L로 간다.
- x는 a와 가까이 가나 x가 a일때는 고려되지 않는다.
- f(x)는 a부근에서 정의 되어 있어야 한다.
- a자체는 빠져도 되나 근처에 있는 값들은 정의 되야 한다.
- 좌극한과 우극한이 다르다면 불연속 점이며, 극한 값이 존재 하지 않는다고 한다.

# One Sided Limits
![image](https://img1.daumcdn.net/thumb/R1280x0/?scode=mtistory2&fname=https%3A%2F%2Fblog.kakaocdn.net%2Fdn%2FdsHH29%2FbtqCOgkKhiH%2FyLKoLIDkxun057vSII5bZ1%2Fimg.jpg)
- 한쪽방향으로 다가갔을때 극한 값이다.
- 좌극한과 우극한이 다르면 불연속 점이며, 극한 값이 존재하지 않는다.

# Precise Definition of a Limit
![](https://mblogthumb-phinf.pstatic.net/20160427_8/abcdduddnd_1461746041213zdqeA_PNG/epsilon-delta.png?type=w2)
- ε>0
- 0< |x-a| < δ 이면 |f(x) -L| < ε 일떄 극한이 정의된다.
- 극한의 후보 a일떄 f(a)=L 이고 L의 근방 L-ε, L+ε 구간을 잡은 후 a의 근방(a-δ ~ a+δ)의 함수 값들이 L-ε, L+ε 구간에 있다면 δ가 존재하고 모든 ε에 δ가 있다면 극한이 존재 한다고 한다.
- ε이 작아 질수록 δ이 작아진다.
